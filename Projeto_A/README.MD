# cryp_to

Description. 
The package cryp_to is used to:
	- encrypt passwords
	- Decrypt passwords

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install cryp_to

```bash
pip install cryp_to
```

## Usage

```python
from cryp_to import cryp_to, cryp_turn
cryp_to.crypS()
cryp_turn.crypS()
```
